# Dotfiles

My personal dotfiles, powered by
[Dotbot](https://github.com/anishathalye/dotbot).

## Usage

```bash
git clone --recurse-submodules https://github.com/jakemarsden/dotfiles.git ~/.dotfiles
~/.dotfiles/install
source ~/.profile
```
